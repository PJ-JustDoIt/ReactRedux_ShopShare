App provides a way to create and update a shared shopping list between family members. 

Features: 
----------
A user can add items they need to the shopping list . 
A cart is created when they buy the items. 
Users can update the shopping list removing items from it that are bought. 



App uses :
-----------
React , Redux , Express ( Node ) , MongoDB, React-bootstrap , html , css


To run the app : 
----------------
- In command lines , run nodemon , webpack . Also needs mongod to be run .
